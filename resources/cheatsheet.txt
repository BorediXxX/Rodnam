Arrays	
	ListArray<Data Type> creates array
	.size returns length of array
	.add(index, value)
	.get(index)

HashMaps
	HashMap<DatatypeKey, DatatypeValue>	creates dictionary
	.size returns length	
	.put(key, value)
	.get(key) return value
	.keySet() returns list of keys
